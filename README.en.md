# vtAdmin

#### Description
vben admin 一款很优秀的ant vue前端框架，但对于一个后端人员开发，不能拿开直接开干，多少有点望而却步。vtAdmin是使用vben admin(前端)+thinkphp6(后端）快速搭建管理后台，旨在让更多后端开发人员快速上手使用vben admin这样优秀的前端框架。
